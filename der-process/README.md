# Lesetagebuch: Kafka - Der Process

Mein Lesetagebuch zu Franz Kafka - Der Process.

## Development

### Dependencies

- `node@20`
- `pnpm`

### Install dependencies

```bash
pnpm install
```

### Run development server

```bash
pnpm dev
```

### Run preview server

```bash
pnpm build
pnpm start
```
